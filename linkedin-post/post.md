I am excited to share my recent hands-on AWS Cloud project where I implemented an **AI-Powered Log Anomaly Detection System using AWS VPC Architecture**.

Project Highlights:

- Configured a secure **Custom VPC** with 2 Public and 2 Private Subnets.
- Developed a high-performance **Java-based Anomaly Detector** using statistical Z-Score analysis.
- Implemented automated log collection and synchronization between **EC2** and **Amazon S3**.
- Secured the infrastructure using a **Bastion Host** and **NAT Gateway** for private subnet isolation.
- Applied **IAM Least Privilege** principles for all resource interactions (no hardcoded keys).
- Visualized system operations with detailed **Mermaid Flowcharts** and activity timelines.

This project helped me strengthen my understanding of:

- **AWS Networking:** VPC, Subnets, Route Tables, IGW, and NAT Gateway.
- **Compute & Storage:** EC2 Instance management and S3 Bucket security.
- **AI & Analytics:** Statistical inference and outlier detection logic in Java.
- **Security:** Bastion hosts, Security Groups, and IAM Roles.
- **Automation:** Bash scripting for end-to-end lifecycle orchestration.

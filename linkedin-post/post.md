# 🚀 New Portfolio Project: AI-Powered Log Anomaly Detection on AWS

I'm excited to share my latest project: a production-ready **AI-Powered Log Anomaly Detection System** built with a secure AWS VPC architecture! 🛡️💻

As a Cloud/DevOps Engineer, I wanted to build something that bridges the gap between secure infrastructure and intelligent operations. This system doesn't just store logs—it analyzes them in real-time to detect threats like CPU spikes and brute-force attempts using statistical Z-Score analysis.

### 🏗️ Key Highlights:

- **Zero-Trust VPC Architecture:** Isolated private subnets, NAT Gateways, and Bastion host for maximum security.
- **High-Performance AI:** A custom-built Java inference engine that processes logs faster than traditional scripts.
- **Full Automation:** End-to-end Bash orchestration for log syncing (S3) and build pipelines.
- **Least Privilege:** Strict IAM policies ensuring secure access without hardcoded keys.

### 🛠️ Tech Stack:

- **Cloud:** AWS (VPC, EC2, S3, IAM, CloudWatch)
- **Language:** Java 17 + Maven
- **Scripting:** Bash Shell
- **Analytics:** Z-Score Statistical Inference

This project pushed me to think deeply about infrastructure-as-code and proactive security monitoring.
